# 隐私政策

**最后更新日期：2026年7月2日**

BossSay（以下简称"本插件"）非常重视用户的隐私保护。本隐私政策说明了本插件如何收集、使用和保护您的信息。

## 数据收集与存储

### 本地存储的数据

本插件使用浏览器的 `chrome.storage.local` API 在您的设备本地存储以下数据：

- **个人资料**：您手动输入的简历内容、个人经历、技能标签、GitHub 链接、作品集链接、自我介绍
- **API 配置**：您配置的 AI 服务 API 地址、API Key、模型名称
- **使用偏好**：消息风格偏好、发送模式设置
- **历史记录**：生成的消息记录（仅存储在本地）

**所有上述数据仅存储在您的浏览器本地，不会上传到任何第三方服务器。**

### 数据外发

当您使用 AI 生成功能时，本插件会将以下内容发送到**您自行配置的 AI API 服务**：

- 您的简历和个人经历内容
- 当前岗位的职位描述（JD）信息

此数据传输仅用于 AI 生成个性化消息，不会被本插件用于其他任何目的。数据传输的目标地址完全由您在设置中自行配置。

### 不收集的信息

本插件**不会**收集以下信息：

- 您的浏览历史
- 您的 Boss 直聘账号密码
- 您与其他用户的聊天内容
- 您的个人身份信息（姓名、电话、邮箱等）
- 任何设备信息或位置信息

## 第三方服务

本插件支持接入第三方 AI 服务（如 DeepSeek、通义千问、智谱 AI 等）。当您使用这些服务时，您的数据将受相应服务的隐私政策约束。建议您在使用前阅读相关服务的隐私政策。

## 数据安全

- 所有数据存储在您的浏览器本地，受浏览器安全机制保护
- API Key 仅存储在本地，不会被传输到除您配置的 AI 服务之外的任何地方
- 本插件不包含任何追踪代码、广告 SDK 或数据收集组件

## 数据删除

您可以随时通过以下方式删除所有数据：

1. 在插件设置页面点击「清除所有数据」
2. 在浏览器扩展管理页面卸载本插件
3. 在浏览器设置中清除扩展数据

# BossSay v4.0.0

**Boss直聘 AI Agentic 智能打招呼助手**

> 基于 ReAct 引擎的多步推理链 + 四层记忆系统 + 自适应学习，分析 JD 匹配简历，生成个性化打招呼消息。

---

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| 🧠 **AI Agentic 引擎** | ReAct 循环（思考→行动→观察→反思），置信度评分，自适应重试 |
| 🔧 **工具注册表** | 6 个内置工具（分析/匹配/评估/生成/审查/修正），可扩展 |
| 📋 **执行规划** | 自动创建步骤计划，依赖追踪，失败跳过 |
| 💾 **四层记忆系统** | 短期记忆 / 长期记忆 / 情景记忆 / 语义记忆 |
| 📚 **自适应学习** | 从发送/回复结果自动学习，优化风格和策略推荐 |
| 🧪 **A/B 测试** | 对比不同风格效果，自动推荐最优方案 |
| 🎯 **目标追踪** | 设定求职目标（回复率/面试数/发送量），实时进度 |
| 🔍 **智能扫描** | 搜索页批量提取，详情页单个提取，全字段可编辑 |
| 💬 **聊天监控** | 自动检测 HR 回复，Chrome 通知提醒 |
| 📊 **数据洞察** | 自动生成回复率分析、趋势报告、风格效果对比 |
| 📄 **PDF 简历** | 上传 PDF 自动提取信息，支持扫描件 OCR（需视觉模型） |
| 🎨 **四种风格** | 专业正式 / 热情亲切 / 幽默轻松 / 简洁明了 |
| ⌨️ **快捷键** | Ctrl+Shift+B 打开 / Ctrl+Shift+G 快速生成 |
| 💾 **备份恢复** | 导出/导入 JSON，可排除 API Key 或简历 |

## 🚀 快速开始

### 安装

1. 下载本仓库（Code → Download ZIP）并解压
2. 打开 `chrome://extensions/` 或 `edge://extensions/`
3. 开启「开发者模式」→「加载已解压的扩展程序」

### 配置 AI 模型

1. 点 BossSay 图标 → ⚙️ 设置
2. 填 API 配置：
   - **推荐**：DeepSeek V4 Flash（`https://api.deepseek.com/v1`，模型 `deepseek-v4-flash`）
   - 通义千问 Turbo（`https://dashscope.aliyuncs.com/compatible-mode/v1`，模型 `qwen-turbo`）
   - 智谱 GLM-4（`https://open.bigmodel.cn/api/paas/v4`，模型 `glm-4-flash`）
   - 任何 OpenAI 兼容 API
3. 💾 保存 → 🔗 测试连接

### 使用

1. 打开 Boss直聘搜索页
2. 点右下角 **BossSay** 浮动面板（或按 `Ctrl+Shift+B`）
3. 🔍 扫描 → 修正字段 → ✨ 生成 → 📝 填入

### 推理链流程

```text
Step 1: 规划 — 分析岗位，回忆相关记忆，制定策略
Step 2: 分析 JD + 匹配简历（1 次 API 调用）
  → 提取核心要求、关键技能
  → 匹配简历中的相关经历
  → 计算匹配度分数 + 置信度
Step 3: 评估匹配度（本地计算）
  → 高匹配(≥70%): 直接展示技能
  → 中匹配(40-70%): 强调可迁移技能
  → 低匹配(<40%): 突出通用能力
Step 4: 生成消息 + 自我审查（1 次 API 调用）
  → 基于匹配度决定策略
  → 按三段式结构生成初稿
  → 检查编造、长度、空洞表达
Step 5: 修正（如审查发现问题，1 次 API 调用）
  → 根据审查意见修正
Step 6: 反思 — 评估最终结果，存储到记忆
```

每步 30 秒超时，推理链全程追踪，置信度可视化，进度实时反馈。

## 💾 记忆系统

```text
┌─────────────────────────────────────────────┐
│              四层记忆架构                     │
├─────────────────────────────────────────────┤
│  短期记忆 (STM)    │ 当前会话上下文，50 条   │
│  长期记忆 (LTM)    │ 持久化存储，500 条      │
│  情景记忆          │ 完整交互记录，90 天衰减  │
│  语义记忆          │ 自动提取的模式和规律     │
├─────────────────────────────────────────────┤
│  公司偏好 │ 角色模式 │ 风格统计 │ 技能组合   │
├─────────────────────────────────────────────┤
│  学习流水线：                                │
│  生成 → 发送 → 回复 → 统计 → 推荐           │
└─────────────────────────────────────────────┘
```

- **自适应风格推荐**：根据公司/职位/历史数据推荐最优风格
- **公司知识库**：记录与每家公司的交互历史和回复率
- **模式识别**：自动发现"什么风格对什么公司有效"
